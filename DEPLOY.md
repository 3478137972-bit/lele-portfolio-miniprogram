## 🚀 微信开发者工具部署步骤

### 步骤 1：打开微信开发者工具

1. 在电脑端打开 **微信开发者工具**
2. 导入项目：`/root/.openclaw/workspace_manager/lele-portfolio-miniprogram`
3. 填写 AppID（测试号或正式号）

### 步骤 2：清除缓存并重新编译

```
工具 → 清除缓存 → 清除全部缓存
重新编译
```

### 步骤 3：全面测试

#### 测试路径 1：自媒体运营折线图
```
底部导航 → 案例集 → 运营相关案例 → 自媒体运营
```
- ✅ 折线图正常显示
- ✅ 点击数据点显示 Tooltip（日期、粉丝数）
- ✅ 点击空白区域隐藏 Tooltip

#### 测试路径 2：社群运营数据卡片
```
底部导航 → 案例集 → 运营相关案例 → 社群运营
```
- ✅ 3 个数据卡片显示（55 人、90%+、4 期）
- ✅ 5 条学员对话反馈
- ✅ 活动类型列表

#### 测试路径 3：讲师介绍
```
底部导航 → 关于
```
- ✅ 讲师信息显示「24 年至今」

### 步骤 4：上传代码

1. 点击右上角 **上传** 按钮
2. 填写版本信息：
   - **版本号**：如 `1.0.0`
   - **版本描述**：如「新增粉丝增长折线图、社群运营数据卡片」
3. 点击 **上传**

### 步骤 5：提交审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **版本管理**
3. 选择刚上传的开发版本
4. 点击 **提交审核**
5. 填写审核信息：
   - **功能页面**：案例集、关于
   - **测试账号**：（如有需要）
   - **补充说明**：作品集展示小程序

### 步骤 6：等待审核

- **审核时间**：通常 1-3 个工作日
- **审核结果**：微信会发送模板消息通知

### 步骤 7：发布上线

1. 审核通过后，登录微信公众平台
2. 进入 **版本管理**
3. 点击 **发布** 按钮
4. 小程序正式上线！

## 🔧 常见问题

### Q1: 图片加载失败（500 错误）
**解决**：检查图片路径，确保文件存在

### Q2: Canvas 不显示
**解决**：清除缓存 → 重新编译 → 查看 Console 日志

### Q3: 分类切换后图表不显示
**解决**：代码已修复，切换时会自动重绘

### Q4: 推送 GitHub 失败
**解决**：网络问题，稍后重试或使用代理
