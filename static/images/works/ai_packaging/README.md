## 上传步骤

### 方法一：通过 Git 命令行（推荐）

1. 将图片文件复制到此目录
2. 在小程序项目根目录执行：
```bash
git add static/images/works/ai_packaging/
git commit -m "feat: 添加 AI 包装设计作品"
git push origin main
```

### 方法二：通过 GitHub Desktop

1. 将图片文件复制到此目录
2. GitHub Desktop 会自动检测到新文件
3. 填写提交信息："feat: 添加 AI 包装设计作品"
4. 点击 Commit & Push

### 方法三：通过 GitHub 网页

1. 访问 GitHub 仓库
2. 导航到 `static/images/works/ai_packaging/` 目录
3. 点击 "Add file" → "Upload files"
4. 拖拽图片文件上传
5. 填写提交信息并提交

## 文件命名规范

建议使用有意义的文件名，例如：
- `packaging-design-001.jpg`
- `ai-baozhuang-case-01.png`
- `product-mockup-2024-03.jpg`

避免使用中文文件名和特殊字符。
