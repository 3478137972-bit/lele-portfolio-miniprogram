# 底部导航图标说明

## 需要的图标（8 个）

请在以下位置下载或创建图标（尺寸：81px × 81px，PNG 格式）：

### 首页图标
- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）

### 作品集图标
- `portfolio.png` - 作品集图标（未选中）
- `portfolio-active.png` - 作品集图标（选中）

### 课程图标
- `course.png` - 课程图标（未选中）
- `course-active.png` - 课程图标（选中）

### 联系图标
- `contact.png` - 联系图标（未选中）
- `contact-active.png` - 联系图标（选中）

---

## 图标获取方式

### 方式 1：从 iconfont 下载（推荐）

1. 访问：https://www.iconfont.cn
2. 搜索关键词：
   - 首页：home, 主页
   - 作品集：gallery, portfolio, 作品
   - 课程：course, 学习
   - 联系：contact, 联系
3. 下载 PNG 格式，尺寸 81×81px
4. 选中状态的图标可以用相同图标但颜色改为橙色 (#F97316)

### 方式 2：使用 emoji 临时替代

如果暂时找不到图标，可以：
1. 先用纯色图标或 emoji 截图替代
2. 后续再替换成正式图标

### 方式 3：用设计工具制作

使用 Figma、Sketch、Photoshop 等工具制作简单的线性图标

---

## 图标颜色建议

- **未选中状态**：#999999（灰色）
- **选中状态**：#F97316（橙色品牌色）

---

## 临时解决方案

如果暂时没有图标，可以：
1. 用任意 81×81 的 PNG 图片临时替代
2. 或者在 `app.json` 中先注释掉 `tabBar` 配置
3. 后续再添加图标
